1 本地两个分支的合并？
   在需要合并的本地分支处 使用命令 git merge [需要合并进来的分支]
   
2 在服务器建立分支
	在需要上传的本地分支处 使用命令 git push origin [需要新建的远程分支xx]
	注意：分支推送到origin主机的xx分支。如果xx不存在，则会被新建。
	
３ 怎么选择远程分支下拉到本地
	git pull <远程主机名> <远程分支名>:<本地分支名>
    
4  删除 untracked files
    git clean -f
   连 untracked 的目录也一起删掉
   git clean -fd
   
5  删除本地分支
   git branch -D br
